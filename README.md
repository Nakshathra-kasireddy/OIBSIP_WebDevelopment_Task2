# 🌐 Personal Portfolio Website

## 📌 Task Overview
This project is a **Personal Portfolio Website** created using HTML and CSS.  
It is designed to showcase personal information, skills, projects, and contact details in a structured and visually appealing format.

---

## 🎯 Objective
The objective of this project is to:
- Build a personal portfolio using HTML and CSS
- Learn how to structure a multi-section webpage
- Practice layout design and styling
- Showcase skills, projects, and contact information in a professional format

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- VS Code (Editor)

---

## 📚 Features
- Header section with profile introduction
- Profile image section
- About me section
- Skills section
- Projects showcase section
- Contact information section
- Clean layout with proper alignment and spacing
- Responsive and visually appealing design

---

## 🚀 What I Learned
- How to design a structured personal portfolio
- How to use HTML sections effectively
- CSS styling for layout, spacing, and alignment
- Creating visually balanced web pages
- Improving UI design skills
